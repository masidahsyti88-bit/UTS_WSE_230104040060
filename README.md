## Deskripsi
Project ini merupakan RESTful API yang menerapkan:
- CRUD lengkap
- HTTP Status Code standar
- Validasi Input
- 7 Prinsip RESTful API
- Data disimpan sementara dalam file .js (tanpa database)

## Cara Menjalankan
1. Clone repository
2. Jalankan perintah:

npm install
npm run dev

3. Akses API di: http://localhost:3000/

## Endpoint API
| Method | Endpoint | Deskripsi |
|------|---------|-----------|
| GET | /api/students | Menampilkan semua data |
| GET | /api/students/:id | Menampilkan data berdasarkan id |
| POST | /api/students | Menambah data mahasiswa |
| PUT | /api/students/:id | Mengupdate data mahasiswa |
| DELETE | /api/students/:id | Menghapus data mahasiswa |
| GET | /api/info | Informasi API |
